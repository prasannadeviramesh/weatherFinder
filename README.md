# Weather Finder üå¶Ô∏è

Weather Finder is a simple and modern weather web app that lets you check **real-time weather details for any city** in the world.  
It is built using **only HTML, CSS, and JavaScript**, without any frameworks or backend.

The app focuses on a clean UI, smooth animations, and an easy user experience.

---

## What this app does

- Shows live weather information for any city
- Displays temperature, humidity, wind speed, pressure, and feels-like value
- Changes background and animations based on weather (sun, rain, clouds, night)
- Saves recent searched cities so you can check them again easily
- Works smoothly on both mobile and desktop screens

---

## Technologies used

- HTML  
- CSS (animations, glassmorphism, responsive design)  
- JavaScript (Fetch API, DOM manipulation, localStorage)  
- OpenWeatherMap API  
- Font Awesome icons  

---

## How to run the project

1. Download the project or copy the HTML file  
2. Open `index.html` in any browser  
3. Enter a city name and click **Get Weather**  
4. Weather details will appear instantly  

No installation or setup is needed.

---

## API usage

This app uses the **OpenWeatherMap API** to get weather data.

Inside the JavaScript code:

```js
const API_KEY = 'YOUR_API_KEY';
```
Replace it with your own API key from:
[OpenWeatherMap API](https://openweathermap.org/api)
